# SPEC: ABDSynthsWeb - Premium Virtual Instruments Boutique

## 🎯 Vision & Objective
Build a high-end web platform for **ABD Virtual Instruments**, inspired by the technical and aesthetic soul of the Stitch prototypes, but industrialized into a professional, high-performance Next.js application.

The goal is to create a "Tech-Noir" digital experience that feels like using high-end hardware (Teanage Engineering, Moog) rather than a generic website.

## 🎨 Aesthetics & Design System (Integrated: Uncodixfy)
- **Concept**: Cyber-Industrial / Precision Engineering.
- **Color Palette**: 
  - Background: `#0e0e0f` (Void Space)
  - Primary: `#00f0ff` (Cyan Neon / Bloom)
  - Surface: `#1a1a1c` (Brushed Metal)
  - Text: `#e5e2e3` (Off-white) / `#849495` (Technical Grey)
- **Typography**: 
  - Headlines: **Space Grotesk** (Wide, technical feel).
  - UI/Labels: **Inter** (High readability).
  - Data: Monospace for technical specs.
- **Constraints (Uncodixfy Standard)**:
  - NO generic glassmorphism. Use "Precision Glass" (subtle, high blur, no borders or very sharp 1px borders).
  - NO bouncy animations. Use "Mechanical" transitions (fast, precise, linear-out-slow-in).
  - NO rounded corners > 8px. Keep it sharp and professional.

## 🏗️ Architecture & Stack
- **Framework**: Next.js 14/15 (App Router).
- **Styling**: Tailwind CSS (Modularized architecture).
- **Animations**: Framer Motion (Synchronized with "Audio Engine" simulations).
- **Methodology**: Superpowers V2 (Spec -> Plan -> Execution -> Verification).

## 🧩 Integrated Skills (from ABDSKILLS)
1. **Superpowers Orchestrator**: Default development workflow.
2. **Uncodixfy**: Design language gatekeeper.
3. **React Best Practices**: Performance and security standard.
4. **Implementation Planning**: Atomic task management.

## 🗺️ Sitemap & Features
1. **Home (The Engine)**: Interactive hero featuring "Neural Matrix_01".
2. **Catalog (The Library)**: Grid of instruments (Junio 601, Omega, Modular Core) with real-time-like visualizer thumbnails.
3. **Instrument Detail**: 
   - 3D-like technical UI.
   - Spectral Analytics (SVG-based visualizers).
   - Technical Specification Matrix.
4. **Download Hub**: Clean, centralized manager for VST3/AU/CLAP installers.

## 🛡️ Iron Laws for this Project
1. **High-Fidelity Assets**: No placeholders. All images must be generated or extracted from the Stitch project and optimized.
2. **Performance First**: LCP < 1.2s. Use Next.js Image optimization and dynamic imports for heavy visualizers.
3. **Responsive Rigor**: Design must feel like a "Pro Tool" on desktop and an "App" on mobile.
