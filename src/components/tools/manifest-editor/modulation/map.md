# Domain: Modulation
Ingeniería de ruteo de señales internas del módulo.

## Componentes
- [ModulationGrid.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/modulation/ModulationGrid.tsx): Matriz interactiva de ruteo cruzado. Permite vincular cualquier fuente con cualquier destino mediante una interfaz de rejilla industrial.
- [ModulationCell.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/modulation/ModulationCell.tsx): Celda atómica de la matriz. Gestiona la intensidad del ruteo mediante interacción de scroll.
