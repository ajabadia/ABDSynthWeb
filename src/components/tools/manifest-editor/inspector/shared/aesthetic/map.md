# Domain: Inspector / Shared / Aesthetic
Componentes modulares de gobernanza estética impulsados por el ElementCatalog.

## Governance Controllers
- [VariantGovernance.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/inspector/shared/aesthetic/VariantGovernance.tsx): Selección de variantes estructurales e industriales (Size_Color pattern o presets de contenedor).

- [TypographyGovernance.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/inspector/shared/aesthetic/TypographyGovernance.tsx): Edición de fuentes, tamaños, colores y alineación de texto.
- [PrecisionGovernance.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/inspector/shared/aesthetic/PrecisionGovernance.tsx): Ajustes de precisión numérica para controles.
