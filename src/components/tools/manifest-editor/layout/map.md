# Domain: Layout
Estructura física y componentes de navegación del workbench.

## Componentes
- [Header.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/layout/Header.tsx): Barra superior principal. Contiene el menú de despliegue, selectores de modo de vista, selector de tema y acceso rápido a la auditoría.
- [WorkbenchSidebar.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/layout/WorkbenchSidebar.tsx): Panel lateral izquierdo. Centraliza las métricas del módulo y el estado de la lógica WASM.
- [WorkbenchFooter.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/layout/WorkbenchFooter.tsx): Barra de estado inferior. Muestra la versión de build, el estado del watchdog de recarga y el badge de cumplimiento.
- [WorkbenchLogs.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/layout/WorkbenchLogs.tsx): Panel de consola colapsable para la monitorización de eventos de ingeniería.
- [ModuleHub.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/layout/ModuleHub.tsx): Contenedor interno del sidebar que organiza las secciones de métricas y activos.
