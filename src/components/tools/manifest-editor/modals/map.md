# Domain: Modals
Gestión de diálogos y procesos de usuario de alto nivel.

## Componentes
- [EditorModals.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/modals/EditorModals.tsx): Orquestador de modales que gestiona la visibilidad y el montaje de los distintos diálogos.
- [AuditModal.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/modals/AuditModal.tsx): Informe de inspección técnica. Proporciona feedback sobre el cumplimiento de estándares y permite exportar el certificado de conformidad.
- [IngestionModal.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/modals/IngestionModal.tsx): Wizard de importación de archivos. Permite seleccionar y clasificar archivos `.acemm`, `.wasm` y recursos adicionales.
- [MockupModal.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/modals/MockupModal.tsx): Estudio de renderizado fotográfico. Genera capturas de alta fidelidad del módulo para documentación comercial.
- [HelpModal.tsx](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components/tools/manifest-editor/modals/HelpModal.tsx): Centro de documentación integrado para usuarios e ingenieros SDK.

## Datos de Soporte
- [helpData.ts](file:///d:/desarrollos/ABDSynthsWeb/abd-ia_synths/src/components\tools\manifest-editor\modals\helpData.ts): Fuente de verdad para el contenido del manual de ayuda.
