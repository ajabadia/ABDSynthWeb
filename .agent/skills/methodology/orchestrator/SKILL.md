# Superpowers Orchestrator (ABD Workflow)

## 🎯 Objetivo
Guiar el desarrollo de ABDSynthsWeb a través de fases rigurosas para asegurar calidad de "grado de ingeniería".

## 🛠️ Workflow (El Ciclo de Vida)

### Fase 1: Concepción (Brainstorming)
- Produce la `SPEC.md` en `.brain/specs/`.
- Define la lógica de negocio y la estética.

### Fase 2: Planificación (Planning)
- Produce el `implementation_plan.md` y el `task.md` en `.brain/`.
- Descompone en tareas atómicas.

### Fase 3: Ejecución (Elite Execution)
- Realiza commits atómicos por cada tarea.
- Sigue el estándar de código de `01-core-code.md`.

### Fase 4: Verificación (Walkthrough)
- Crea el `walkthrough.md`.
- Verifica contra los diseños originales de Stitch.

## ⚖️ Leyes de Hierro
- Prohibido codear sin Plan.
- Prohibido saltarse la Verificación.
