# Uncodixy UI Standard (ABD Synths Edition)

Este skill define la estética innegociable para ABDSynthsWeb. El objetivo es alejarse del "look de IA" y acercarse a la precisión de la ingeniería de audio de alta gama.

## 🎨 Estética Tech-Noir
- **Background**: Deep Black `#0e0e0f`.
- **Primary**: Electric Cyan `#00f0ff`.
- **Surface**: Dark Zinc / Brushed Metal `#1a1a1c`.

## ⚖️ Leyes de Hierro Visuales
1. **Bordes**: Máximo 8px. Nada de botones tipo "pastilla" o burbuja.
2. **Vidrio (Glass)**: Solo permitido con `backdrop-filter: blur(24px)` y opacidad mínima (<5%). Debe parecer cristal de laboratorio.
3. **Sombras**: Prohibidas las sombras difusas de colores. El relieve se logra con bordes de 1px de color sólido (`border-white/10`).
4. **Tipografía**:
   - `Headline`: Space Grotesk (Wide / Tech).
   - `Body`: Inter (Clean).
   - `Data`: JetBrains Mono / System Mono.

## 🚫 Hard No (Prohibido)
- NO a los gradientes suaves de fondo.
- NO a las animaciones "bouncy" o elásticas.
- NO a los componentes de UI genéricos sin modificar.
