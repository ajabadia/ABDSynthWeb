## Standard View Angles ([VIEW_ANGLE])
For a complete set, generate 4 separate images for each key:
1. **FRONT**: "Straight frontal eye-level view."
2. **TOP**: "Cenital 90-degree top-down view."
3. **BACK**: "Straight rear view showing ports and industrial labels."
4. **DIAGONAL**: "Perspective top-diagonal 3/4 view from a 45-degree angle."
