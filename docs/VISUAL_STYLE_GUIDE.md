# ABD Virtual Instruments: Visual Style Guide

## Master Rendering Prompt
To ensure a consistent "Boutique" look across the entire catalog, use the following prompt template for all instrument renders.

### The Template
> "A spectacular, hyper-realistic cinematic 3D render of the [INSTRUMENT_NAME] synthesizer shown in the reference image. The synthesizer is placed in a dark, professional high-end music studio. Dramatic perspective view from a slightly low angle. Every fader, slider, and button from the reference image must be in its exact original position. Materials: [PANEL_MATERIAL], authentic [SIDE_PANEL_MATERIAL]. Lighting: Subtle electric cyan glow from the screen and indicators, and soft magma orange glow from the active LEDs. Cinematic depth of field with a slightly blurred dark studio background. High resolution, 8k, photorealistic textures."

### Applied Variables for JUNiO-601
- **INSTRUMENT_NAME**: ABD JUNiO-601
- **PANEL_MATERIAL**: brushed black aluminum front panel
- **SIDE_PANEL_MATERIAL**: dark walnut wooden side panels

### Applied Variables for NEURONiK
- **INSTRUMENT_NAME**: ABD NEURONiK
- **PANEL_MATERIAL**: aseptic white matte polymer front panel
- **SIDE_PANEL_MATERIAL**: polished carbon fiber side panels
